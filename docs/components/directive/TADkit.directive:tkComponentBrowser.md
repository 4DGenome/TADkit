



# TADkit.directive:tkComponentBrowser








Genome track browser directive
Generates tracks using the user specified browser
(see Browsers module)








## Directive Info


* This directive executes at priority level 0.


## Usage




* as element:
    ```
    <ta-dkit-directive-tk-component-browser>
    ...
    </ta-dkit-directive-tk-component-browser>
    ```

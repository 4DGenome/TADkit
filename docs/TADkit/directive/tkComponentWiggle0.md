



# tkComponentWiggle0








Track component which generates a d3 graph
from supplied BigWig (wiggle format) data.








## Directive Info

* This directive creates new scope.
* This directive executes at priority level 0.


## Usage




* as element:
    ```
    <tk-component-wiggle0>
    ...
    </tk-component-wiggle0>
    ```
* as attribute:
    ```
    <ANY>
    ...
    </ANY>
    ```

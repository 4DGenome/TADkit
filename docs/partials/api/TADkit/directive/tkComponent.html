tkComponent

  1. - directive in module TADkit

Dummy components directive that is replaced on complie by real component directive from supplied object type. e.g. from a array of components objects

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <tk-component>
...
</tk-component>
  • as attribute: 
    <ANY>
...
</ANY>

Example

<div tk-component ng-repeat='component in components'></div>





# ui.directive:uiTooltip








UI Tooltip.
use ng-cloak or style="display:none" on tag to ensure hidden onLoad








## Directive Info

* This directive creates new scope.
* This directive executes at priority level 0.


## Usage




* as element:
    ```
    <ui-directive-ui-tooltip>
    ...
    </ui-directive-ui-tooltip>
    ```
* as attribute:
    ```
    <ANY>
    ...
    </ANY>
    ```
